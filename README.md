# Jetpack Compose Internal

<img width="340" height="340" alt="character" src="https://github.com/user-attachments/assets/8e1d4bd1-8d10-42c5-a73b-1a19faf15ea3" />

이 스터디의 목적은 컴포즈 내부에 대해 더 잘 이해하고 학습하고 또한 많은 **낙오자**를 발생시킨 이 책을 끝까지 읽기 위함입니다. <br>
발표가 서툴러도 OK! 다른 사람들 보다 못 하면 어떡하지?라는 걱정은 NO! <br>
**부담없이 편안하게 끝까지 같이 읽어 봅시다!** <br>
책은 [이곳](https://leanpub.com/composeinternalskor)에서 구매하실 수 있고 300 페이지 조금 안 됩니다.
### 진행 방식
- **매주 일요일 오후 8시 구글 밋업**으로 카메라를 켜고 서로 얼굴을 보며 진행(1시간 ~ 1시간 30분 정도 예상)
  -  스터디는 녹화되어, 해당 기수 스터디원에게만 제공
- 발표자는 [랜덤 핀볼](https://lazygyu.github.io/roulette/)로 1 ~ 2명 선정
- 발표자는 공부한 내용을 화면 공유를 통해 발표
- 발표가 끝나면 Q&A 진행 및 추가 내용 공유
- 만든 질문 리스트를 공유하며 같이 고민해보기(개수는 자유, 1개도 OK)
- 공부한 내용은 깃헙 레포 fork 후 PR하기
  -  작성 방식은 자유롭게
- 스터디 외 궁금한 것들은 issue로 등록하여 스터디 기수 관계없이 모두 자유롭게 의견 공유

## 이 스터디를 참여하신 분들의 후기
**1기** <br>
"혼자했을 때 끝까지 진행하지 못하였던 책 내용을 스터디 덕분에 끝까지 진행할 수 있게 되어서 너무 좋았습니다."

"내부 코드까지 깊게 들여다볼 수 있는 기회가 있어서 좋았습니다."

"이번 기회를 통해 컴포즈를 전반적으로 이해하기 위해서는 한 번쯤 꼭 짚고 넘어가야 하는 과정이라는 것을 알게 되었습니다. 모든 부분을 완벽하게 이해하긴 어려웠던 스터디여서 나중에 기회가 된다면 한 번 더 참여하면 좋을 것 같습니다."

"묵혀둔 숙제를 한 느낌."

"부담없이 진행할 수 있어서 좋았습니다.
다른분이 준비한 스터디 내용을 참고해서 내가 잘못알고 있었던 부분도 잡을 수 있었고
인사이트도 많이 얻을 수 있어서 좋았습니다"

"번역본이라 하더라도 말의 표현이나 개념 설명이 어려운 부분이 있는데 다같이 학습 내용을 공유하면서 진행하다보니 더 이해가 쉬웠고 많이 배운것 같습니다." <br>

**2기** <br>
"혼자 읽을 때보다 훨신 수월하게 Jetpack Compose Internals 책을 읽을 수 있습니다.
모임장님께서 많은 자료를 공유해주셔서 학습효율이 좋았습니다.
아직 읽지 않으셨다면 꼭 참여해보세요, 강력 추천합니다. 👍👍"

"컴포즈 내부원리 스터디를 참여하시면 읽기 어려워 방치한 컴포즈 내부원리를 완독 할 수 있습니다."

"N회독 하고 싶은 책과 N회차 참여하고 싶은 스터디의 조합입니다. 젯팩 컴포즈 인터널스를 구매하고 한 번도 열어보지 않은 주변 지인들에게 적극 추천하고 있습니다. 만약 여기에 해당하신다면 이번 기회에 완독해보세요."

"혼자만의 의지로 도전하기 어려웠던 책에 스터디라는 강제성을 부여해 끝까지 읽을 수 있었습니다. 또, 각자 이해하고 정리한 내용이 달라 다른 분들의 글을 보며 제가 생각지 못한 부분을 학습할 수도 있었습니다. 무엇보다 스터디장님이 굉장히 열정 넘치십니다! 이전에 정리했던 내용이나 유용한 자료도 많이 공유해주셨어요. 그래서 완독하는데 있어 가장 많은 동기부여를 얻었습니다. 감사합니다!😄"

**이 스터디를 참여하신 모든 분이 친구, 동료, 지인분들께 이 스터디를 추천한다고 응답하였습니다.**
